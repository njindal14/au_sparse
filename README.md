To run, simply clone the repo and run root -l crossSectionsAuAu.C, 
making sure to change the paths to where you have your tracking and 
TOFMatch efficiencies, the Run10 data file, and the FemtoPairFormat.h file. 
You will also want to create the output_root_files subdirectory. 
Otherwise this will run out of the box. I've included my file that calculates
my pairwise efficiencies from simulation as well.
